# Sidecar themes

Sidecar ships with five persistent shell themes:

- **Codex Green** — near-black surfaces, muted green panels, and terminal-green text.
- **Codex Dark** — neutral charcoal surfaces with a restrained green accent.
- **Midnight** — deep navy surfaces and violet accents inspired by the Sidecar artwork.
- **Light** — high-contrast light surfaces.
- **System** — resolves to the Windows app light/dark preference at launch.

The theme applies only to the native Sidecar shell. The embedded ChatGPT page controls its own appearance. The selected theme is stored under `%LOCALAPPDATA%\ChatGPTSidecar\Settings\ui.json`.
